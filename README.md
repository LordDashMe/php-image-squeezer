# PHP Image Squeezer

A simple PHP package for image compression powered by FFMPEG.

## Requirement(s)

- PHP version from 5.4.* up to latest.

## Install

### via Composer

- Use the command below to install the package via composer:

```txt
composer require lorddashme/php-image-squeezer
```
